## Goal
Create reusable Input, Textarea, Modal, and Avatar components in `client/src/components/ui/`.

## Tasks

### Task 1: Create Input component
**File**: `client/src/components/ui/Input.jsx`
**What**: Styled input with label, optional icon, error state.
**Props**: `label`, `error`, `icon` (lucide component), `className`, plus all native input props via ...rest
**Styling**: `w-full px-4 py-2.5 rounded-xl border border-slate-200 bg-white text-sm focus:ring-2 focus:ring-teal-500/20 focus:border-teal-500 outline-none transition`
**Error state**: Red border + error text below input.
**Icon**: Positioned left inside input with pl-10.

### Task 2: Create Textarea component
**File**: `client/src/components/ui/Textarea.jsx`
**What**: Same styling as Input but for multi-line text.
**Props**: `label`, `error`, `className`, `rows` (default 3), plus ...rest
**Key**: `resize-none` to prevent user-caused overflow.

### Task 3: Create Modal component
**File**: `client/src/components/ui/Modal.jsx`
**What**: Centered overlay modal with close button, title, and body.
**Props**: `open`, `onClose`, `title`, `children`, `maxWidth` (default 'max-w-lg')
**Implementation**:
- Portal rendered via createPortal to document.body
- Backdrop: fixed inset-0, bg-black/50, onClick closes
- Content: centered with `fixed inset-0 flex items-center justify-center z-50 p-4`
- Panel: `bg-white rounded-2xl shadow-xl w-full {maxWidth} max-h-[90vh] overflow-y-auto`
- Title bar with X close button
- Uses rc-scale-in animation class
- Escape key closes modal (useEffect with keydown listener)
- Body scroll lock when open

### Task 4: Create Avatar component
**File**: `client/src/components/ui/Avatar.jsx`
**What**: User avatar with gradient fallback (uses existing getAvatarGradient from utils/brand).
**Props**: `name`, `src` (optional image URL), `size` (sm/md/lg/xl), `className`
**Sizes**: sm=h-8 w-8, md=h-10 w-10, lg=h-12 w-12, xl=h-16 w-16
**Behavior**:
- If `src` provided: render img with object-cover, rounded-full
- If no `src`: render div with gradient background + first letter of name
- Always rounded-full

### Task 5: Create barrel export
**File**: `client/src/components/ui/index.js`
**What**: Re-export all UI primitives for clean imports.
```js
export { default as Card } from './Card';
export { default as Button } from './Button';
export { default as Badge } from './Badge';
export { default as Input } from './Input';
export { default as Textarea } from './Textarea';
export { default as Modal } from './Modal';
export { default as Avatar } from './Avatar';
```

## Acceptance Criteria
- [ ] Input renders with label, focus ring, and error state
- [ ] Textarea renders with resize-none and consistent styling
- [ ] Modal opens centered, closes on backdrop click, closes on Escape, animates in
- [ ] Modal z-index is above everything (z-50)
- [ ] Avatar shows gradient fallback with initial when no image
- [ ] Avatar shows image when src is provided
- [ ] All primitives importable from `components/ui`
- [ ] No overflow issues on any component
